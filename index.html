CovidInfo

26 May 2021

CovidInfo is an API based website designed to bring user the latest info on global Covid-19 statistics. This is a part of CodingBN assignments, where the participants were assigned to build an API based website. So our team decided to use Covid-19 API from Rapid API website. Here are some of the features for this project:

  • A world map - divided into 6 continents which can be navigate using the navbar
  • Statistics for each continent (confirmed case, and death count)
  • Each country on the map will be color coded based on total confirmed case per population
  • You can click on the map to see the full statistics of the selected country, or you can use the search option
  • Current Global Statistics
  • Charts showing the top 10 country for each of these cases - total confirmed case, total death, new case, and total recovered case
